# 词的黏着

## 否定式

### 名词

在词后面加上「じゃない」即可。

- 学生（がくせい）じゃない
    不是学生
- 元気（げんき）じゃない
    不好。

### 形容词

#### な形容词

同名词，在词后面加上「じゃない」即可。

- 静か（しずか）じゃない
    不安静。
- 好き（すき）じゃない
    不喜欢。

#### い形容词

首先把结尾的「い」去掉，然后加上「くない」。

- 高い（たかい） -> 高くない
- 美味しい（おいしい）-> 美味しくない

#### 例外

- いい -> よくない
- かっこいい -> かっこよくない

实际上「いい」是由「良い（よい）」演变而来，所以会有这种不规则的否定式。

### 动词

#### る动词

把「る」去掉换成「ない」。

- 食べる（たべる）-> 食べない

#### う动词

1. 对以「う」结尾的う动词：把「う」换成「わ」，再加上「ない」。
    - 買う（かう）-> 買わない
2. 其他う动词：把末假名换成同行的あ段假名，再加上「ない」。
    - 待つ（まつ）-> 待たない

#### 例外

1. する -> しない
2. くる -> こない
3. *ある -> *ない

## 过去时

### 名词

1. 在名词后面加上「だった」。
    - 学生（がくせい）-> 学生だった
2. 对于否定式的过去时，首先将名词变为否定式，再将「じゃない」末尾的「い」去掉，再加上「かった」。
    - 友達じゃない（ともだちじゃない）-> 友達じゃなかった

### 形容词

#### な形容词

同名词。

- 好きじゃなかった（すきじゃなかった）
- 静かじゃなかった（しずかじゃなかった）

#### い形容词

首先把结尾的「い」去掉，然后加上「かった」。

- 高い（たかい） -> 高かった
- 美味しじゃない（おいしじゃない）-> 美味しじゃなかった

#### 例外

- いい -> よかった
- よくない -> よくなかった
- かっこいい -> かっこよかった

### 动词

#### る动词

把「る」去掉换成「た」。

- 食べる（たべる）-> 食べた

#### う动词

1. す -> した
    - 話す（はなす） -> 話した
2. く -> いた
    - 書く（かく） -> 書いた
3. ぐ -> いだ
    - 泳ぐ（およぐ）-> 泳いだ
4. む/ぶ/ぬ -> んだ
    - 飲む（のむ）-> 飲んだ
    - 遊ぶ（あそぶ）-> 遊んだ
    - 死ぬ（しぬ）-> 死んだ
5. る/う/つ -> った
    - 切る（きる）-> 切った
    - 買う（かう）-> 買った
    - 持つ（もつ）-> 持った

#### 动词的否定式的过去时

先将动词变为否定式，再将「ない」中的「い」换成「かった」。

- 捨てる（すてる）-> 捨てなかった
- 行く（いく）-> 行かなかった

#### 例外

1. する -> した
2. くる -> きた
3. 行く（いく） -> 行った


## 动词词根（stem）

在一般日语课本里把它称为ます词根（masu-stem）。

### る动词

把「る」去掉。

- 食べる（たべる） -> 食べ

### う动词

把最后的假名从う段换成同行的い段假名。

- 泳ぐ（およぐ） -> 泳ぎ

### 例外

- する -> し
- くる -> き

## て形（te-form）

### 名词和な形容词

在词后面加上「で」。

- 静か（しずか） -> 静かで
- 一般的（いっぱんてき） -> 一般的で

### い形容词、否定式名词或形容词

把「い」换成「くて」。

- 狭い（せまい） -> 狭くて
- 彼女（かのじょ）じゃない -> 彼女じゃなくて
- いい -> よくて

### 动词

- 肯定式：把动词活用成过去形，然后把「た」换成「て」，或把「だ」换成「で」。
- 否定式：跟い形容词一样，把「い」换成「くて」。

以上规则也适用于以「です」和「ます」结尾的叮咛语。

## ている

「ている」表示动作进行时。

- 読む（よむ）
    - 読んでいる 在读
    - 読んでいた 当时在读
    - 読んでいない 不在读
    - 読んでいなかった 当时不在读

注意「知る（しる）」和「分かる（わかる）」。它们的动作进行时表示一个已经知道了或者已经明白了的状态，而一般时表示“知道”或“明白”这个动作。

- 今日（きょう）、知（し）りました。我今天才知道(我今天做了知道这个动作)
- この歌を知っていますか？(你)知道这首歌吗?

「行く（いく）」和「来る（くる）」同理。

### てある

「てある」表示已经发生的结果状态。这个词通常用来解释某件事已经处在完成的状态，并且还带着点为另外一件事做好了准备的意思。

- 準備（じゅんび）はどうですか。准备的如何了？
- 準備は、もうしてあるよ。已经准备好了。

### ておく（とく）

「ておく」显示声明某个动作已经完成（或即将完成），并且为未来做好了准备。「ておく」有时可以简写成「とく」。

1. 晩ご飯（ばんごはん）作っておく（つくっておく）。
    - 做晚饭（准备吃）。
2. 電池（でんち）買っておきます（かっておきます）。
    - 我要买些电池（以后用）。

### ていく、てくる

把「行く」和「来る」与动作的て形合用，这时候表示这个动作朝向或者来自某个地方。

- 鉛筆（えんぴつ）を持っている（もっている）。
    - （你）有铅笔吗？
- 鉛筆を学校（がっこう）へ持っていく。
    - （你）要带铅笔去学校吗？
- 鉛筆を家に持ってくる。
    - （你）要带铅笔回家吗？

除此之外，「ていく」可以表示未来的趋势，而「てくる」可以表示迄今为止的情况。

- 一生懸命（いっしょうけんめい）、頑張（がんば）っていく！
    - (未来)会尽全力的!
- 色々（いろいろ）な人（ひと）と付き合ってきた（つきあってきた）けど、いい人はもだ見つからない（みつからない）。

## 可能形（potential form）

### る动词

将「る」换成「られる」。

- 見る -> 見られる

### う动词

将最后的假名从う段改为同行的え段假名，再加上「る」。

- 游ぐ -> 泳げる

注意，日语里表示可以看到的、可以听见的是一般不用可能形而是用现成的词：「見える（みえる）」和「聞こえる（きこえる）」。

### 例外

- する -> できる
- くる -> こられる
- ある -> あり得る（ありえる/ありうる）

## 情态动词

### 不能做什么

将动词活用为て形，加上助词「は」，再加上「だめ」、「いけない」或者「ならない」。

- 入る（はいる）
    - 入ってはだめ
    - 入ってはいけない
    - 入ってはならない
- 食べる（たべる）
    - 食べてはだめ
    - 食べてはいけない
    - 食べてはならない

若要使用它们的敬语，那先把「いけない」和「ならない」还原为「いける」和「なる」后再改为敬语的否定式即可。

### 必须做什么

以下三种方式皆可表示必须做：

1. 否定式的て形+助词「は」+だめ/いけない/ならない
2. 动词的否定式+表示条件的「と」+だめ/いけない/ならない
3. 动词的否定式+表示条件的「ば」+だめ/いけない/ならない

- 毎日学校に行かなくてはなりません。
- 毎日学校に行かないとだめです。
- 毎日学校に行かなければいけません。

### 简化的说法

上述三种表示必须的表达存在口语的简化说法：

- 把「なくて」换成「なくちゃ」并省略后面的だめ/いけない/ならない
- 把「なければ」换成「なきゃ」并省略后面的だめ/いけない/ならない
- 直接使用「と」并省略后面的だめ/いけない/ならない

否定的表达也存在口语的简化说法：

- 把「ては」变成「ちゃ」但后面的だめ/いけない/ならない不能省略
- 把「では」变成「じゃ」但后面的だめ/いけない/ならない不能省略

### 表达可以或不可以做什么

在动词的て形后面加上「も」即可，后面一般跟「いい」、「大丈夫」或「構わない」。

- 全部（ぜんぶ）食べ（たべ）てもいいよ。

## 希望

### 「たい」

对于希望做的动作，在动词词根后面加上「たい」即可。加上「たい」后的词的变形跟い形容词一致，但是与い形容词不同的是它后面可以接「を」、「に」、「へ」或「で」等助词。

- 行く（いく）
    - 行きたい
    - 行きたくない
    - 行きたかった
    - 行きたくなかった

### 「欲しい」

「欲しい（ほしい）」可以用来表示想要什么东西。但它也可以和动词的て形一起使用表示想要完成什么事情，此时一般写成「ほしい」

- 大きい縫いぐるみが欲しい。
- 部屋を綺麗にしてほしいのよ。

## 意向形

### 口语里的普通提议做某事

#### る动词

去掉「る」再加上「よう」。

- 食べる -> 食べよう

#### う动词

把末尾假名从う段改为同行お段，再加上「う」。

- 入る -> 入ろう

### 叮咛语提议做某事

在词根后面加上「ましょう」即可。

- 食べる -> 食べましょう
- 入る -> 入りましょう

## 尝试

### 尝试某个东西

将动词活用为て形后加上「みる」。组合起来的动词的活用规则和「見る」一致。

- 切る（きる）
    - 切って見る
    - 切ってみた
    - 切ってみない
    - 切ってみなかった

### 尝试做什么

把动词活用为意向形后加上「とする」。

- 見る -> 見ようとする
- 行く -> 行こうとする

这里的「する」也可以换成「決める」表示「决定做某事」或「思う」表示「想做某事」。

## 授受动词

### 「あげる」、「やる」

「あげる」是站在说话人⻆度，表示「给」的意思。当你想给别人东西，或者想为别人做什么的时候就必须用这个动词。

要表示为想为别人做什么的话，在做的动词的て形后面加上「あげる」即可。

「やる」和「あげる」意思类似，但一般给的对象是动物。

### 「くれる」

「くれる」是另一个表示「给」的东西，不过与「あげる」不同，它是用在接受者身上的。别人给你什么，或者为你做什么的时候就必须用这个动词(其实就是「あげる」的反方向)。

要表示别人为你做了什么时同理，在做的动词的て形后加上「くれる」。

### 「もらう」

「もらう」是接受的意思，也可以接在て形动词后面。常与「から」、「に」搭配。

### 用「くれる」或「もらえる」请别人帮忙

「もらえる」是「もらう」的可能形。这两者意思类似，但是主语宾语是反过来的。

- あなたが、私に千円を貸してくれる（かしてくれる）？
- 私が、あなたに千円を貸してもらえる。

### 让别人别做什么

让别人别做什么，只要在动词否定式后面加上「で」即可。

## 做请求

### 「ください」

「ください」表示礼貌请求，相比「くれる」强硬，可以与请求的东西或动词て形连用。口语时可以直接去掉「ください」只保留て形。对于需要比「くれる」更加强硬的语气的场景，可以使用「くれる」但把最后的「る」去掉。

「ください」和「ます」一样只出现在句尾，不能用在从句修饰名词，直接引用（*需加引用连接*）除外。

### 「ちょうだい」

口语里「ください」的替代词是「ちょうだい」，比较女性化和孩子气。把「ちょうだい」写成汉字「頂戴（ちょうだい）致す（いたす）ます」的时候一般用在非常正式的表达里。语法上和「ください」类似。

### 「なさい」

「なさい」是一种温和礼貌但坚决的命令方式，通常用于母亲教训小孩或者老师老师教训学生。「なさい」只能用在肯定意的动词后面。和动词一起使用时在动词词根后加上「なさい」。

### 命令形

动词可变为命令形对人下命令。日常生活里的日本人一般很礼貌，很少对人使用命令形下命令。

### る动词

把「る」换成「ろ」。

- 食べる -> 食べろ

### う动词

把末尾假名从う段改为同行え段。

- 死ぬ（しぬ） -> 死ね

### 否定命令形

把「な」加在动词后面即可。
