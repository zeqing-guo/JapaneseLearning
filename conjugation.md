# 词的黏着

## 否定式

### 名词

在词后面加上「じゃない」即可。

- 学生（がくせい）じゃない
    不是学生
- 元気（げんき）じゃない
    不好。

### 形容词

#### な形容词

同名词，在词后面加上「じゃない」即可。

- 静か（しずか）じゃない
    不安静。
- 好き（すき）じゃない
    不喜欢。

#### い形容词

首先把结尾的「い」去掉，然后加上「くない」。

- 高い（たかい） -> 高くない
- 美味しい（おいしい）-> 美味しくない

#### 例外

- いい -> よくない
- かっこいい -> かっこよくない

实际上「いい」是由「良い（よい）」演变而来，所以会有这种不规则的否定式。

### 动词

#### る动词

把「る」去掉换成「ない」。

- 食べる（たべる）-> 食べない

#### う动词

1. 对以「う」结尾的う动词：把「う」换成「わ」，再加上「ない」。
    - 買う（かう）-> 買わない
2. 其他う动词：把末假名换成同行的あ段假名，再加上「ない」。
    - 待つ（まつ）-> 待たない

#### 例外

1. する -> しない
2. くる -> こない
3. *ある -> *ない

## 过去时

### 名词

1. 在名词后面加上「だった」。
    - 学生（がくせい）-> 学生だった
2. 对于否定式的过去时，首先将名词变为否定式，再将「じゃない」末尾的「い」去掉，再加上「かった」。
    - 友達じゃない（ともだちじゃない）-> 友達じゃなかった

### 形容词

#### な形容词

同名词。

- 好きじゃなかった（すきじゃなかった）
- 静かじゃなかった（しずかじゃなかった）

#### い形容词

首先把结尾的「い」去掉，然后加上「かった」。

- 高い（たかい） -> 高かった
- 美味しじゃない（おいしじゃない）-> 美味しじゃなかった

#### 例外

- いい -> よかった
- よくない -> よくなかった
- かっこいい -> かっこよかった

### 动词

#### る动词

把「る」去掉换成「た」。

- 食べる（たべる）-> 食べた

#### う动词

1. す -> した
    - 話す（はなす） -> 話した
2. く -> いた
    - 書く（かく） -> 書いた
3. ぐ -> いだ
    - 泳ぐ（およぐ）-> 泳いだ
4. む/ぶ/ぬ -> んだ
    - 飲む（のむ）-> 飲んだ
    - 遊ぶ（あそぶ）-> 遊んだ
    - 死ぬ（しぬ）-> 死んだ
5. る/う/つ -> った
    - 切る（きる）-> 切った
    - 買う（かう）-> 買った
    - 持つ（もつ）-> 持った

#### 动词的否定式的过去时

先将动词变为否定式，再将「ない」中的「い」换成「かった」。

- 捨てる（すてる）-> 捨てなかった
- 行く（いく）-> 行かなかった

#### 例外

1. する -> した
2. くる -> きた
3. 行く（いく） -> 行った


## 动词词根（stem）

在一般日语课本里把它称为ます词根（masu-stem）。

### る动词

把「る」去掉。

- 食べる（たべる） -> 食べ

### う动词

把最后的假名从う段换成同行的い段假名。

- 泳ぐ（およぐ） -> 泳ぎ

### 例外

- する -> し
- くる -> き

## て形（te-form）

### 名词和な形容词

在词后面加上「で」。

- 静か（しずか） -> 静かで
- 一般的（いっぱんてき） -> 一般的で

### い形容词、否定式名词或形容词

把「い」换成「くて」。

- 狭い（せまい） -> 狭くて
- 彼女（かのじょ）じゃない -> 彼女じゃなくて
- いい -> よくて

### 动词

- 肯定式：把动词活用成过去形，然后把「た」换成「て」，或把「だ」换成「で」。
- 否定式：跟い形容词一样，把「い」换成「くて」。

以上规则也适用于以「です」和「ます」结尾的叮咛语。

## ている

「ている」表示动作进行时。

- 読む（よむ）
    - 読んでいる 在读
    - 読んでいた 当时在读
    - 読んでいない 不在读
    - 読んでいなかった 当时不在读

注意「知る（しる）」和「分かる（わかる）」。它们的动作进行时表示一个已经知道了或者已经明白了的状态，而一般时表示“知道”或“明白”这个动作。

- 今日（きょう）、知（し）りました。我今天才知道(我今天做了知道这个动作)
- この歌を知っていますか？(你)知道这首歌吗?

「行く（いく）」和「来る（くる）」同理。
